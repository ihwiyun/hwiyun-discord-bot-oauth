개인정보처리방침

최종 수정일: 2025년 12월 10일

1. 수집하는 정보

본 봇은 YouTube 구독 인증을 위해 다음 정보를 수집합니다:

  • Discord 사용자 ID: 서버에서 사용자를 식별하기 위해
  • YouTube 채널 ID: 사용자가 직접 입력한 채널 정보
  • OAuth 2.0 액세스 토큰: YouTube 구독 목록 확인용 (암호화되어 저장)
  • 인증 일시: 인증 기록 관리용

2. 정보 사용 목적

수집한 정보는 다음 목적으로만 사용됩니다:

  • YouTube 구독 여부 확인
  • Discord 서버에서 역할 자동 부여
  • 인증 기록 관리

3. 정보 보관 기간

다음과 같이 보관됩니다:

  • 인증 정보: 사용자가 인증을 해제할 때까지
  • OAuth 토큰: 사용자가 인증을 해제하거나 토큰이 만료될 때까지

4. 정보 보호

수집한 정보는 안전하게 보관되며, 다음 조치를 취합니다:

  • OAuth 토큰은 암호화되어 저장
  • 서버에만 저장되며 외부에 공개되지 않음
  • 제3자와 공유하지 않음

5. 사용자 권리

사용자는 다음 권리를 가집니다:

  • 인증 정보 삭제 요청
  • 인증 해제 (Discord에서 /인증해제 명령어 사용)

6. 문의

개인정보 처리에 대한 문의사항이 있으시면 Discord 서버로 연락해주세요.
