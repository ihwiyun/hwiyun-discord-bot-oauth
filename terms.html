서비스 약관

최종 수정일: 2025년 12월 10일

1. 서비스 이용

본 봇은 Discord 서버에서 YouTube 구독 인증 서비스를 제공합니다.

서비스를 이용함으로써 본 약관에 동의하는 것으로 간주됩니다.

2. 사용자 의무

사용자는 다음을 준수해야 합니다:

  • 정확한 정보 제공
  • 서비스 남용 금지
  • 타인의 계정 사용 금지
  • Discord 서버 규칙 준수

3. 서비스 제공

본 서비스는 다음과 같이 제공됩니다:

  • YouTube 구독 인증
  • 치지직 팔로워 인증
  • 자동 역할 부여

4. 서비스 중단

다음 경우 서비스가 중단될 수 있습니다:

  • 서비스 점검
  • 기술적 문제
  • 정책 위반

5. 면책 조항

본 서비스는 "있는 그대로" 제공되며, 다음과 같은 경우 책임을 지지 않습니다:

  • 서비스 중단으로 인한 손해
  • 사용자의 부주의로 인한 손해
  • 제3자의 행위로 인한 손해

6. 약관 변경

본 약관은 사전 고지 없이 변경될 수 있습니다.

7. 문의

서비스 약관에 대한 문의사항이 있으시면 Discord 서버로 연락해주세요.
